# Oblivion - clothing shop

E-commerce multipage application with connected database and payment system.

`<link>` : <https://oblivionshop.netlify.com/>

## Stack

- HTML5
- SASS
- React
- Redux
  - Redux-persist
  - Redux-saga
- Firebase
- Stripe
